# Job Market Intelligence CLI

A Python command-line application that analyzes job market trends, roles, salaries, and demand using real-world data.

## 🚀 Features

- Search job listings
- Analyze market demand
- View salary insights
- Store data with SQLite
- Interactive CLI interface

## 🛠 Tech Stack

- Python
- SQLite
- APIs
- CLI

## ▶️ How to Run

```bash
python main.py
